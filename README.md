# 💰 Fintrack — Fintech Dashboard

A modern fintech dashboard built using **React + TypeScript + Zustand + Tailwind CSS**, designed with a clean, production-style UI.

## ✨ Features

### 📊 Dashboard

* Total balance, income, and expenses overview
* Monthly balance trend (line chart)
* Spending breakdown (pie chart)

### 💳 Transactions

* View, search, and filter transactions
* Edit transaction (modal)
* Delete transaction
* Color-coded income & expenses

### 🔐 Role-Based Access

* Admin → full access (edit/delete)
* Viewer → read-only mode

### 📈 Insights

* Total income & expenses
* Highest expense tracking
* Data-driven summaries

---

## 🧠 Tech Stack

* React (Vite) + TypeScript
* Zustand (State Management)
* Tailwind CSS
* Recharts
* Vercel (Deployment)

---

## ⚙️ Run Locally
